# Recipe App 🍳

## ■ 概要

このアプリは、料理のレシピを記録・管理・検索できるWebアプリです。
ユーザーは材料や調味料、分量、写真を登録し、自分だけのレシピ集を作成できます。
また、登録されたレシピを材料や調味料から検索することが可能です。

将来的には、他のユーザーが投稿したレシピの検索や共有機能も実装予定です。

---

## ■ 目的

日々の料理において「家にある材料で何が作れるか分からない」という課題を感じ、このアプリを開発しました。
また、バックエンド開発力（特にデータベース設計・API設計）を向上させることを目的としています。

## ■ 主な機能

### 現在実装予定

* ユーザー登録・ログイン機能
* レシピ投稿機能

  * 写真
  * 材料・分量
  * 調味料・分量
* レシピ一覧表示
* 材料・調味料による検索機能

### 今後実装予定

* 他ユーザーのレシピ閲覧
* お気に入り機能
* タグ検索
* AND検索（複数材料）
* レコメンド機能
